### 輸送問題
工場 (x, y) から商品を店 (a, b, c) に配送します。供給量、需要量、輸送コストが下表で与えられているとき、総輸送コストが最小となる配送の仕方を求めてください。  

- 表 : 輸送コスト  
| |店 a|店 b|店 c|供給量|  
|---|---|---|---|---|  
|工場x|10|6|16|8|   
|工場y|8|8|4|16|  
|需要量|12|4|8| |
